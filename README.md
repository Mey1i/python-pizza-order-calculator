# Python Pizza Order Calculator

A simple Python console program that calculates the final price of a pizza order based on size, pepperoni, and extra cheese options. The user selects the pizza size (S/M/L), chooses toppings, and the script returns the final bill. Includes basic input validation and clear pricing logic.

## Features
- Choose pizza size (S, M, L)
- Optional pepperoni and extra cheese
- Automatic bill calculation
- Input validation for all choices

1. Install Python 3
2. Save the script as `pizza.py`.
